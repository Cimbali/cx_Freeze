About cx_Freeze

cx_Freeze is a set of scripts and modules for freezing Python scripts into executables, in much the same way that py2exe and py2app do. Unlike these two tools, cx_Freeze is cross platform and should work on any platform that Python itself works on. It supports Python 3.5.2 or higher (for Python 2.7 use cx_Freeze version 5.1.x).

Download/Install

Version 6.3, released October 20, 2020

Install by issuing the command 

pip install cx_Freeze --upgrade
or 
python3 -m pip install cx_Freeze --upgrade
or download directly from PyPI.
